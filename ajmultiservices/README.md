# AJ Multiservices — Template-style Vite + React site

Top contact bar → nav → hero banner → services cards → about strip → gallery → reviews → quote form → footer.

## Run
```bash
npm i
npm run dev
```

## Customize
- Replace `public/hero.jpg` and `public/about.jpg`
- Replace gallery images in `public/work/*`
- Update phone/email/location in `src/App.tsx`
- Wire quote form to backend in `onSubmit`
